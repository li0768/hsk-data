# HSK中文文本难度分级预测器

一个基于深度学习的中文文本HSK难度级别预测工具，能够自动判断中文文本对应的HSK等级（1-9级）。

## 特性

- 🚀 **高精度预测**: 基于优化的深度学习模型，准确率超过94%
- 📚 **多级别支持**: 支持HSK1到HSK9所有级别
- 🔧 **简单易用**: 提供简洁的API接口
- 📊 **批量处理**: 支持批量文本预测
- 💾 **多种输出**: 支持JSON、CSV等多种输出格式
